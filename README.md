# Magic Voice Changer

A production-oriented Flutter architecture template for a prank voice calling app inspired by FunCall.

## Stack
- Flutter + Dart 3
- Riverpod + GoRouter
- WebRTC services abstraction
- Firebase integration hooks
- Google Play Billing and AdMob-ready package setup

## Run
```bash
flutter pub get
flutter run
```

## Notes
This scaffold includes production-grade architecture boundaries (core/services/features/shared) and UI for all requested screens.
